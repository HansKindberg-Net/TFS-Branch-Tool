TFS-Branch-Tool
===============
This is a fork of the code from [Quick Response Sample – Visual Studio Extension to support the ALM Rangers Branching and Merging guidance - Visual Studio ALM + Team Foundation Server Blog - Site Home - MSDN Blogs](http://blogs.msdn.com/b/visualstudioalm/archive/2012/11/22/quick-response-sample-visual-studio-extension-to-support-the-alm-rangers-branching-and-merging-guidance.aspx). The source is downloaded from [Supporting Guidance and Whitepapers - Download: Quick Response Sample Solutions](http://vsarguidance.codeplex.com/releases/view/96222), direct download-link: http://vsarguidance.codeplex.com/downloads/get/523297. This branch contains the original code downloaded 2013-02-04.

Requirements
------------
Requirements to open/build the solution:
* **Visual Studio Ultimate 2012** (fakes are used)
* **Microsoft Visual Studio 2012 SDK** (TOOLS -> Extensions and Updates... and search for "Microsoft Visual Studio 2012 SDK")
